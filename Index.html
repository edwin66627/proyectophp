"CRUD" CREAR, LEER, MODIFICAR & ELIMINAR

CREATE, READ, UPDATE & DELETE

Aplicativo realizado para ver, agregar, actualizar, y eliminar registros con PHP y MySQL

Instrucciones:

  1. Ir a la opcion ver.
  2. Agregar elementos desde el boton agregar.
  3. Seleccionar el boton Editar de cualquier elemento.
  4. Seleccionar el boton Eliminar de cualquier elemento.

  • Facil de utilizar y versionar
  • Ideal para plataformas web
  • Efectivo para crear un contacto mas cercano con sus clientes
  • Conserva la informacion de los clientes de una manera mas organizada
